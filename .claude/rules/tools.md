# MCP 도구 개발 규칙

## 도구 정의

- 모든 도구 함수에 명확한 독스트링 작성 (Args, Returns 필수)
- 입력 파라미터에 Pydantic 모델 또는 타입 힌트 사용
- MCP 호환성을 위해 JSON 직렬화 가능한 반환값 사용

## AWS SDK 사용

- boto3 클라이언트는 함수 레벨에서 생성 (전역 금지)
- 리전 설정은 환경 변수에서 로드
- 페이지네이션이 필요한 API는 paginator 사용

## 보안

- IAM 권한은 최소 권한 원칙 적용
- 민감한 데이터는 반환값에서 마스킹
